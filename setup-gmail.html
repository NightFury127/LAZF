Setup Gmail for OTP

Enter your Gmail credentials to enable sending OTP emails.

Important Security Note

For Gmail, you need to use an "App Password" instead of your regular password. This is a 16-character code that Google generates for apps that don't support 2-factor authentication.

To create an App Password:

  1. Go to your Google Account
  2. Select Security
  3. Under "Signing in to Google," select App Passwords (you may need to sign in again)
  4. At the bottom, choose Select app and select "Other (Custom name)"
  5. Enter "Lazreus Tech" and click Generate
